# AGENTS.md

## Objetivo

Este repositório contém apenas o portfólio pessoal de Bruno Rendeiro. Apresenta o autor e liga para aplicações externas. O código das outras aplicações nunca deve ser adicionado aqui.

## Stack e comandos

- React, TypeScript e Vite
- `npm run dev`: desenvolvimento local
- `npm run check`: validação de tipos
- `npm run build`: build de produção

## Organização

- `src/data/apps.ts`: fonte única dos projetos apresentados
- `src/App.tsx`: páginas e componentes da interface
- `src/styles.css`: sistema visual responsivo

## Regras para agentes

- Manter cada app externa num repositório independente.
- No portfólio, guardar apenas metadados, imagens e links das apps.
- Não adicionar backend sem uma necessidade explícita.
- Preservar acessibilidade, responsividade e português europeu.
- Executar `npm run check` e `npm run build` depois de alterações de código.
